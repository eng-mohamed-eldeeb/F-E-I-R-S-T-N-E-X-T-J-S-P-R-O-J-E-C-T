## Getting Started

To get a local copy up and running follow these simple example steps.

`git clone https://github.com/eng-mohamed-eldeeb/simpleNextJSAPP.git `

Then `cd simpleNextJSAPP`

Then `npm install` to install packages

Finally `npm run dev`
